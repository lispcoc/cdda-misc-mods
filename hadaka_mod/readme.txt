# 裸MOD

裸、あるいは行き過ぎた超軽装を強化する要素を追加します。
バニラ環境では裸や超軽装でのプレイは非常に困難ですが、
本MODの追加要素を活用すれば裸でもニューイングランドを跋扈するモンスターたちとも戦える…
ぐらいのバランスを目指します。

## 追加特質
・持たざる者
服を脱げば脱ぐほど戦闘能力が上がります。逆に服を着ても特別弱くなることはありません。

・裸体主義者
ランダムで着ている服を脱ぐことがあり、嫌でも裸に近い恰好で生活することになります。
脱ぐ確率は部位と被覆率に依存します。

・露出狂
胴と両脚の露出度が高いと能力が上がり、低いと大幅に下がります。

## 追加職業
・裸の英雄
世紀末を裸一貫に拳ひとつで生き抜くHENTAIです。縛りは緩いので裸ビギナーにもおすすめできます。

・ヌーディスト
ただただ裸になりたいだけの人です。裸体主義者持ちで所持品もないので生き残れるかは運にかなり左右されます。

・裸の王子様/裸の姫様
異世界からやってきた超軽装の勇者です。初期状態で知性に大幅ペナルティがついているので、うまくカバーする必要があります。
